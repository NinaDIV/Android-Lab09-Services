# Android Lab 09 - Services y Tareas en Segundo Plano

Laboratorio Android sobre el manejo de procesos en segundo plano: Services, WorkManager y notificaciones del sistema.

## Temas cubiertos
- Android Services (Started & Bound)
- WorkManager para tareas diferidas
- Notificaciones del sistema con NotificationCompat
- Permisos en tiempo de ejecucion
